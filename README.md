# MLang

This is the implementation fo a custom language called MLang. The language is build on top of ANTLR4 as the front-end and targets LLVM JIT as the backend.

# Features

- [ ] Arithmetic
- [ ] Boolean operations
- [ ] Printing
- [ ] Branching
- [ ] Loops
- [ ] Strictly typed
- [ ] Integer type
- [ ] Boolean type
- [ ] String type
- [ ] Float type
- [ ] Function type / closures
- [ ] Type inference
- [ ] Structs
- [ ] Functions
